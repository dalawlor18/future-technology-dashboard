How to Use This Dashboard

1 Opening the Dashboard

  1. → Double-click FutureTechnology_GHG_Dashboard.html to open in your browser
  2. → Works in Chrome, Firefox, Edge, or Safari — no installation needed
  3. → Use the tabs at the top to navigate between sections
  4. → Hover over any chart element to see exact values
  5. → Click legend items to show or hide individual data series

2 Updating Charts with New Year Data

  1. → Go to the Update Data tab
  2. → Enter your Scope 1, 2 and 3 totals for the new year (e.g. 2026) in the input table
  3. → Click "Update All Charts"
  4. → All charts across every tab update instantly
  5. → Note: data entered here is not saved when you close the browser — see Step 4 to make permanent updates

3 Updating the Excel File

  1. → Open FutureTechnology_Sustainability_Dashboard.xlsx
  2. → Navigate to the relevant scope sheet (Scope 1, Scope 2 or Scope 3)
  3. → Enter data in the yellow input cells for the new year column
  4. → GHG Summary, Net Zero Tracker and Environment KPIs will auto-update
  5. → Run update_all.py to refresh the embedded Excel charts
python update_all.py

4 Making Permanent Updates to This HTML File

  1. → Open dashboard.py in a text editor
  2. → Find the GHG_DATA dictionary near the top
  3. → Add the new year's values
  4. → Run python dashboard.py to regenerate both the Excel and HTML files
GHG_DATA = {
  2024: {"s1": 19587, "s2_loc": 732, "s3": 3696},
  2025: {"s1": 19576, "s2_loc": 921, "s3": 5249},
  2026: {"s1": YOUR_VALUE, ...}, ← add here
}

5 Understanding the Data

  1. → All emissions in tCO2e (tonnes of CO2 equivalent)
  2. → Scope 1 = direct emissions (gas, vehicles, refrigerants)
  3. → Scope 2 = indirect from purchased electricity and steam
  4. → Scope 3 = value chain (transport, waste, travel, commuting)
  5. → Emission factors from US EPA eGRID 2022 and IPCC AR6
  6. → Grid region: AZNM Arizona (0.000386 tCO2e/kWh)

6 Sharing the Dashboard

  1. → Send the .html file as an email attachment — recipients open it in their browser
  2. → No internet connection required — all charts run locally
  3. → To print: use browser print (Ctrl+P) and select "Save as PDF"
  4. → To host online: upload the .html file to any web server or GitHub Pages
